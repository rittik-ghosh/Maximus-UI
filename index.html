Introduction
Maximus UI is an open source styling framework, which helps Developers to use best styles and components, by using pre-defined classnames.
Maximus helps you build your Project at much ease.
Installation Guide
In order to introduce Maximus to your project, Copy the Link given below and paste it in the tag of your html, above all other style sheets. 
                        
                    <link rel="stylesheet" href="https://https://maximus-ui.netlify.app/Stylesheets/style.css">
