Input
Inputs are necessary for user interaction with the website. Inputs are used to take information from the user, and then save the data to the server, accordingly.
Label Inputs
Label Input has a label present at the top of the input field. They can be customized to show error or success messages, and change color accordingly.
Wrong Password. Try again.
Password Match. 
                        
                <-- Using 'input-group' and variants followed by 'varients' for input div -->

                        <div class="input-group">
                        <label>Username *</label>						
                        <input type="text" placeholder="Please Enter your Username">
                        </div>

                        <div class="input-group input-error">
                        <label>Password *</label>						
                        <input type="password" placeholder="Please Enter your Password">
                        <div class="error-message">
                            Wrong Password. Try again.
                        </div>
                        </div>

                        <div class="input-group input-success">
                        <label>Password *</label>						
                        <input type="password" placeholder="Please Enter your Password">
                        <div class="success-message">
                            Password Match.
                        </div>
                        </div>
Checkbox Input
Checkboxes allow the user to select one or more items from a set. Checkboxes can be used to turn an option on or off.
If you have multiple options appearing in a list, you can preserve space by using checkboxes instead of on/off switches. If you have a single option, avoid using a checkbox and use an on/off switch instead. 
                        
                            <-- Using the 'checkbox-ip' class -->

                            <div class="checkbox-ip">
                                <input id="men-cloathing" name="men" type="checkbox" checked>
                                <label for="men-cloathing" Men Clothing</label>
                            </div>
        
                            <div class="checkbox-ip">
                                <input id="women-cloathing" name="women" type="checkbox">
                                <label for="women-cloathing" women Clothing</label>
                            </div>
        
                            <div class="checkbox-ip">
                                <input id="kids-cloathing" name="kids" type="checkbox">
                                <label for="kids-cloathing" kids Clothing</label>
                            </div>
Radio Input
Radio buttons allow the user to select one option from a set. 
                        
                            <-- Using the class 'checkbox-ip' followed by radio input -->

                        <div class="checkbox-ip">
                            <input id="threeStars" name="rating" type="radio" checked>
                                <label for="threeStars">3 stars</label>
                        </div>
        
                        <div class="checkbox-ip">
                            <input id="twoStars" name="rating" type="radio">
                                <label for="twoStars">2 stars</label>
                        </div>
        
                        <div class="checkbox-ip">
                        <input id="oneStar" name="rating" type="radio" disabled>
                            <label for="oneStar">1 star</label>
                         </div>
Slider Input
The slider input gives the user a wide range of inputs. 
                        
                <-- Using the 'slideContainer' class followed by slider Input class 'slider' -->

                        <div class="slideContainer">
                            <label for="myRange">Range</label>
                            <input id="myRange" name="range" type="range" min="0" max="100" value="50" step="10" class="slider"> 
                        </div>
