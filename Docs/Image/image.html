Image
Responsive Image Components are used to dsiplay Images on websites. They expand upto the width of the container they are in.
Square Image
Square images have an equal height and width. This can be used by asding the calss 'sq-img' 
                        
                    <-- Using the 'sq-img' class name -->

                    <img src="url of the pic" class="sq-img"></img>
Round Image
Round images are circular and can be obtained by adding the class 'round-img'. 
                        
                            <-- Using the 'sq-img' class name -->

                            <img src="url of the pic" class="round-img"></img>
Responsive Image
Responsive Image change their dimensions w.r.t the size of the parent container. 
                        
                            <-- Using the 'responsive-img' class name -->

                            <img src="url of the pic" class="responsive-img"></img>
