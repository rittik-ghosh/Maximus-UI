Avatar
Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognisable visual identity.
Profile Avatar
Profile Avatars can be created using the class 'avatar-circle', inside the img tag. 
                        
                    <-- Using the 'avatar-circle' class name -->

                    <img class="avatar-circle" src="https://url-of-the-image" alt="avatar">
Sizes Avatar
Size of the avatars can be changed by adding avatar-sm, avatar-md, avatar-lg class to the existing avatar class. 
                        
                    <-- Using the avatar-sm | avatar-md | avatar-lg class names -->

                    <img class="avatar-circle avatar-sm" src="https://url-of-the-image" alt="avatar">
                    <img class="avatar-circle avatar-md" src="https://url-of-the-image" alt="avatar">
                    <img class="avatar-circle avatar-lg" src="https://url-of-the-image" alt="avatar">
Square Avatar
Adding a class 'avatar-square' to existing class, will generate a square shaped avatar.
Note : Square class would better work if height & width are same. 
                        
                    <-- Using the square class -->

                    <img class="avatar-square" src="https://url-of-the-image" alt="avatar">
                    <img class="avatar-square" src="https://url-of-the-image" alt="avatar">
                    <img class="avatar-square" src="https://url-of-the-image" alt="avatar">
Shadow Avatar
Adding a class 'shadow' to existing class, will generate a shadow shaped circilar-avatar.
                        
                    <-- Using the shadow class -->

                    <img class="avatar-circle shadow" src="https://url-of-the-image" alt="avatar">
                    <img class="avatar-circle shadow" src="https://url-of-the-image" alt="avatar">
                    <img class="avatar-circle shadow" src="https://url-of-the-image" alt="avatar">
