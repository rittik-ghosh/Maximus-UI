Toast
Toast display messages related to an interaction happened between user and the website. They usually stay on the screen for 2-3 seconds and then disappear.
Toast can be used to show success messages, warnings or errors.
Simple Toast
The toast can be obtained by using the class 'Snack-bar' followed by the variant i.e. the place you want the toast to appear. The variants include bottom-center, bottom-right, bottom-left, top-center, top-right, top-left.

Can't send photo. Retry in 5 seconds.

                        
                        <-- Using the 'snack-bar' class name followed by variant-->
                                <div class="snack-bar">
                                <div class="content">
                                    <p>Can't send photo. Retry in 5 seconds.</p>
                                </div>
                                <div class="btns">
                                <button class="margin-center bgcolor-transparent retry">
                                        RETRY
                                </button>
                                <button class="margin-center bgcolor-transparent close">
                                        X
                                </button>
                                </div>
                                </div>
